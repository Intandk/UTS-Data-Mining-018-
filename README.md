# UTS Data Mining

## Prediksi Kualitas Wine Menggunakan Random Forest Classifier

Project ini dibuat untuk memenuhi tugas UTS mata kuliah Data Mining.

## Dataset
Dataset yang digunakan adalah Wine Quality Dataset yang berisi fitur kimiawi wine dan target quality.

## Model
Model machine learning yang digunakan:
- Random Forest Classifier

## Evaluasi
Evaluasi model menggunakan:
- Accuracy
- Confusion Matrix
- Classification Report
